# React To-Do App

website link: https://mytodo-taskmanager.vercel.app/

A simple and responsive To-Do application built using "React", "Tailwind CSS", and "React Hooks".  
It allows users to add, edit, mark as complete, and delete tasks efficiently â€” all with a clean and minimal UI.  
All tasks are stored in localStorage, ensuring your data stays saved even after refreshing or reopening the browser.

---

##  Features
- Add new tasks quickly
- Edit existing task names
- Mark tasks as completed / uncompleted
- Delete tasks easily
- Persistent storage using localStorage
- Fully responsive design built with Tailwind CSS
- Smooth navigation using React Router
   - Includes Home Page for managing todos
   - Includes About Page describing the app and its purpose

---

- React (Functional Components & Hooks)

- React Router DOM
- JavaScript (ES6+)
